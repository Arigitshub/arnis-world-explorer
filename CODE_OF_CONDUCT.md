# Code of Conduct

## Our Pledge

We want this project to be welcoming, practical, and respectful.

## Our Standards

Examples of positive behavior:

- giving actionable feedback
- respecting different experience levels
- focusing on the code and the problem
- being clear about constraints and tradeoffs

Examples of unacceptable behavior:

- harassment
- insults or personal attacks
- bad-faith review behavior
- deliberately misleading users about what the project can do

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributions
that violate this code of conduct.
